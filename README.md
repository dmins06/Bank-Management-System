# Bank Management System

A database management project that simulates the operations of a commercial bank. The system manages customers, employees, branches, bank accounts, loans, and financial transactions using a relational database. It is designed following database design principles, including ER modeling, normalization, relational schema design, and SQL implementation to ensure data integrity, consistency, and efficient data management.

## Features

* Customer management
* Employee management
* Branch management
* Bank account management
* Deposit, withdrawal, and transfer transactions
* Loan management
* Transaction history
* SQL queries, views, triggers, and stored procedures

## Technologies

* MySQL
* SQL
* ER Diagram (ERD)
* Relational Database Design
* Database Normalization

## Project Goal

To design and implement a secure, efficient, and scalable relational database for managing banking operations while applying database design concepts learned in the Database Systems course.
